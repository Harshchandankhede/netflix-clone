# CineStream — Premium Streaming Landing Page

A beautifully designed, fully responsive landing page for a fictional movie and TV show streaming catalog called **CineStream**. It features a modern, dark-themed UI/UX aimed at newsletter subscriptions and driving user engagement.

## 🚀 Features
- 🎨 **Modern UI/UX Design**: Built with vibrant colors, dark themes, and subtle gradients.
- 📱 **Cross-Device Compatibility**: Fully responsive layout optimized for mobile, tablet, and desktop screens.
- 📰 **Newsletter Subscription Modal**: Interactive, clean modal for gathering email subscriptions without intrusive password fields.
- 🔥 **Dynamic Trending Carousel**: A sleek, horizontal scrolling row displaying trending movies with beautiful typography and hover effects.
- ❓ **Interactive FAQ Accordion**: Smooth, animated accordion for frequently asked questions built entirely with Vanilla JavaScript.
- 🏎️ **Optimized Performance**: Lightweight architecture using pure HTML/CSS/JS without heavy frameworks.

## 🛠️ Tech Stack
- **HTML5** (Semantic structure)
- **CSS3** (Flexbox, Grid, Custom Variables, Animations)
- **Vanilla JavaScript** (DOM manipulation, event listeners, modal logic)
- **Google Fonts** (Inter)

## 💻 Running Locally
1. Clone or download this repository.
2. Open the `index.html` file in your web browser.
3. Enjoy the interface!

## 🛡️ Anti-Phishing Compliant
This project has been explicitly designed to avoid triggering "Deceptive Site Ahead" or phishing warnings on hosting platforms like GitHub Pages. 
- It uses a custom brand (**CineStream**) and a unique color palette.
- It operates strictly as a newsletter subscription landing page.
- **No password fields** or deceptive login forms are used, ensuring safe deployment anywhere.

## 🚀 Deployment
Deploy this static site easily to GitHub Pages, Netlify, or Vercel.

**For GitHub Pages:**
1. Push the repository to GitHub.
2. Go to **Settings > Pages**.
3. Select the `main` branch as your source and hit Save.
4. Your landing page will be live shortly!
